# RenameDataSet

## 工程简介

### C++ win32重新按顺序命名文件名工具

## 工程详解

- **开发环境**
	
  > Win10 64位 + vs2015

- **运行参数** 

	> 运行命令：RenameDataSet.exe .txt book
	
	> 文件名格式：book01.jpg book02.jpg book03.jpg

- **注意**
	
  > 不支持原文件名有中文或中文字符情况
